# 记忆条目模板

完整的记忆条目格式模板，包含所有可选区块。

## 基础模板

```markdown
## [记忆标题]

<!-- @meta category: 类别 | tags: 标签 1, 标签 2 | created: YYYY-MM-DD | updated: YYYY-MM-DD -->

[记忆内容]

支持多段落、列表、表格等 markdown 格式。

### 相关记忆（可选）

**前置信息**
- [[相关记忆标题]](文件名.md#相关记忆标题)

**参考信息**
- [[另一记忆]](另一文件.md#另一记忆)

### 被引用（可选）

- [[引用本条目的记忆]](文件.md#标题) 引用了本条目

<!-- @end -->
```

## 组件说明

### 必填部分

| 组件 | 说明 | 示例 |
|------|------|------|
| `## 标题` | 记忆标题，使用二级标题 | `## 沟通风格偏好` |
| `<!-- @meta ... -->` | 元数据注释 | 见下方 |
| 内容 | 记忆主体内容 | 任意 markdown |
| `<!-- @end -->` | 结束标记 | 固定格式 |

### Meta 格式

```markdown
<!-- @meta category: 类别 | tags: 标签 1, 标签 2 | created: YYYY-MM-DD | updated: YYYY-MM-DD -->
```

| 字段 | 必填 | 说明 |
|------|------|------|
| `category` | 是 | 与文件类别对应 |
| `tags` | 是 | 逗号分隔的标签 |
| `created` | 是 | 创建日期（ISO 格式） |
| `updated` | 是 | 更新日期（ISO 格式） |

### 可选部分

#### 相关记忆

当本记忆与其他记忆有关联时添加：

```markdown
### 相关记忆

**前置信息**
- [[用户基本信息]](user-profile.md#用户基本信息)

**参考信息**
- [[时间管理方法]](knowledge-base.md#时间管理方法)

**后续行动**
- [[年度体检]](tasks-reminder.md#年度体检)
```

#### 被引用

记录哪些记忆引用了本条目（可由工具自动生成）：

```markdown
### 被引用
- [[年度体检]](tasks-reminder.md#年度体检) 引用了本条目
```

## 完整示例

```markdown
## 日本旅行方案

<!-- @meta category: decisions | tags: 旅行，日本，决策 | created: 2026-03-05 | updated: 2026-03-05 -->

**决定内容**：五一假期选择日本关西旅行

**出行人员**：一家三口（夫妻 +5 岁孩子）

**旅行时间**：2026 年 5 月 1 日 -5 月 7 日（7 天 6 晚）

**预算**：人均 15000 元，总计约 45000 元

**已确认事项**：
- [x] 目的地：日本关西（大阪、京都、奈良）
- [x] 住宿：京都 ryokan 酒店 6 晚
- [ ] 航班：待选择
- [ ] 行程：待规划

**决策原因**：
1. 飞行时间短（3 小时），适合带孩子
2. 文化相近，饮食适应
3. 预算适中

### 相关记忆

**前置信息**
- [[家庭信息]](user-profile.md#家庭信息)

**参考信息**
- [[签证申请流程]](knowledge-base.md#签证申请流程)

**后续行动**
- [[签证办理提醒]](tasks-reminder.md#签证办理提醒)

<!-- @end -->
```

## 使用提示

1. **标题唯一性** - 同一文件内标题应唯一
2. **标签一致性** - 使用统一的标签命名
3. **链接准确性** - 确保链接标题与目标完全一致
4. **及时更新** - 修改内容时更新 `updated` 日期
5. **适度关联** - 每条记忆 2-5 个相关链接为宜
