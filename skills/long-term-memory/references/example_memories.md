## 时间管理方法

### 四象限法则

将任务按重要和紧急程度分为四类：

| | 紧急 | 不紧急 |
|---|---|---|
| **重要** | 第一象限：立即做 | 第二象限：计划做 |
| **不重要** | 第三象限：授权做 | 第四象限：不做 |

**应用建议**：
1. 尽量减少第一象限的事情（救火）
2. 把时间投入到第二象限（预防、规划）
3. 第三象限的事情尽量授权他人
4. 避免第四象限的事情（浪费时间）

### 番茄工作法

1. 选择一个任务
2. 设置 25 分钟倒计时
3. 专注工作，不做其他事情
4. 时间到，休息 5 分钟
5. 每 4 个番茄后，休息 15-30 分钟

**适用场景**：
- 需要专注的工作
- 容易分心的时候
- 大任务分解执行
